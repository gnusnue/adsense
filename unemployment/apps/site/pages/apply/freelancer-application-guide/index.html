프리랜서 실업급여 신청 가이드

프리랜서의 경우 직장인과 동일한 흐름으로 판단하면 오해가 생길 수 있습니다. 먼저 적용 대상인지 확인하고, 24개월·180일 기준과 신청 절차를 순서대로 점검하세요.

신청 전 확인 3단계

  1. 적용 대상 여부: 예술인/노무제공자 등 해당 고용보험 적용군인지 확인
  2. 기간 요건: 24개월 내 180일 기준 충족 여부 확인
  3. 이직 사유: 비자발적 이직 또는 인정 가능한 사유인지 확인

신청 절차

  1. 구직등록
  2. 수급자격 신청자 교육 이수
  3. 수급자격 인정 신청
  4. 실업인정 일정 확인
  5. 회차별 활동/소득신고 관리

준비서류 체크

  • 신분증
  • 본인 명의 계좌 정보
  • 이직 관련 확인서류
  • 계약 종료/소득 증빙 자료
  • 추가 요청 대비 원본 자료 보관

연결 가이드

  • 부모 페이지: 실업급여 신청방법
  • 자격 확인: 실업급여 자격 3분 자가진단
  • 소득신고: 알바·단기근로 소득신고 가이드
  • 아카이브: 롱테일 콘텐츠 아카이브

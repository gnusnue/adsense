# 5탭 고정 콘텐츠 운영 가이드

기준일: 2026-02-24
목적: 상단 네비게이션/플로팅 탭은 5개로 고정하고, 기타 콘텐츠는 하위 구조로 관리

## 1) 상단 탭 정책

- 상단 헤더 + 플로팅 탭은 아래 5개만 유지
  - 계산기 (`/`)
  - 신청 방법 (`/apply/`)
  - 자격 조건 (`/eligibility/`)
  - 실업인정 (`/recognition/`)
  - 소득신고 (`/income-report/`)
- `FAQ` 및 기타 상세 콘텐츠는 상단 탭에 추가하지 않음

## 2) 기타 콘텐츠 노출 정책

- 노출 위치는 아래 3곳으로 제한
  - 각 페이지 하단 `관련 가이드` 카드
  - 푸터 링크
  - (향후) 가이드 허브 페이지 `/guides/`
- 원칙: 상단 탭은 정보 탐색의 “뼈대”, 상세 콘텐츠는 본문/허브에서 소비

## 3) 내부링크 규칙

- 상세 콘텐츠는 반드시 핵심 5페이지 중 1개 이상으로 링크
- 핵심 5페이지는 관련 가이드 카드 3개 이상 유지
- 앵커 텍스트는 검색의도형으로 통일
  - 예: `실업급여 자격조건 3분 체크`, `실업인정 회차별 증빙 방법`

## 4) 신규 콘텐츠 추가 기준

- 신규 URL은 아래 중 하나를 만족할 때만 생성
  - 반복 질의(검색/상담) 빈도가 높음
  - 기존 5개 페이지에 넣기 어려운 독립 주제
  - 정책 변경으로 별도 설명이 필요
- 생성 시 필수 요소
  - 상단 즉답(요약 3줄)
  - 공식 출처 2개 이상
  - 계산기 또는 핵심 5페이지 CTA 1개 이상

## 5) 운영 체크리스트

- [ ] 상단 탭이 5개로 유지되는지 확인
- [ ] 신규 콘텐츠가 상단 탭을 늘리지 않는지 확인
- [ ] 관련 가이드/푸터를 통해 접근 가능한지 확인
- [ ] 내부링크가 핵심 5페이지 권위로 수렴하는지 확인
